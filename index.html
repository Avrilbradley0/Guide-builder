제작가이드 HTML 빌더

기존 제작가이드 복사 → 이미지 자동 추출 → 단색/별색/공통 지정 → 3가지 코드 개별 복사

기본 설정 제품명과 툴팁 이미지는 생성되는 HTML 상단에 반영됩니다.
상단 고정 설정

1. 붙여넣기

Ctrl + V
1
기존 제작가이드 페이지에서 이미지 영역 또는 전체 페이지를 복사합니다.
2
아래 박스 클릭 후 Ctrl + V로 붙여넣습니다.
3
이미지 URL만 자동 추출되어 오른쪽 목록에 표시됩니다.
페이지 전체를 복사해도 이미지 URL만 골라냅니다. 이미 추가된 이미지는 중복 추가되지 않습니다.
